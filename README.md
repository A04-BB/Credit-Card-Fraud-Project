# Credit Card Fraud Detection

**GitHub Repository:** https://github.com/A04-BB/Credit-Card-Fraud-Project

## How to Download the Dataset

**STEP 1** — Go to https://www.kaggle.com and create a free account if you do not have one

**STEP 2** — Navigate to the dataset using the following link:

https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

**STEP 3** — Click the Download button and select download dataset as zip

**STEP 4** — Locate the downloaded zip file on your computer and extract it. Inside the extracted folder you will find `creditcard.csv`

**STEP 5** — Move `creditcard.csv` into the data folder in the root of this project. If the folder does not exist, create one called data

**STEP 6** — Confirm the file is located in the data folder

## How to Set Up and Run Notebooks & GUI

Make sure you have Python 3.8 or above

**STEP 1** — Open the project in VS Code and open the terminal

**STEP 2** - Create a virtual environment enter:

python -m venv .venv

**STEP 3** - Activate the virtual environment by:

**Windows:**
.venv\Scripts\activate

**Mac / Linux:**
source .venv/bin/activate

**STEP 4** — Install required libraries

pip install notebook jupyterlab ipykernel streamlit pandas numpy scikit-learn matplotlib seaborn

**STEP 5** — Install VS Code extensions, Python and Jupyter extensions installed

**STEP 6** — Open the project folder in VS Code, open  Data Exploration Notebook, select the kernel and choose `.venv`

**STEP 7** — Run each notebook in order by clicking **Run All** at the top

**STEP 8** – Run this command in the terminal to run the GUI 

**sSTEP 9** – The GUI should build successfully and open up automatically.
